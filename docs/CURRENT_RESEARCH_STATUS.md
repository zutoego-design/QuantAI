  Fama-French 数据和完整 Python 包环境。

  现金利率也成为显式配置。

## 当前 P1 阻断项

以下项目已经进入数据质量 warning 和 run bias flags，但外部数据尚未接入：

- 授权的历史指数成分和公司行动主数据。
- 精确退市收益、并购、分拆、分红、换股和派生证券账本。
- 带修订历史的 PIT 基本面和 ALFRED 宏观 vintage。
- SEC 历史文本覆盖至少 `80%`。
- 第二股票池数据源交叉验证；当前配置仍为
  `validation_provider: disabled`。
- 用真实成交数据校准点差、冲击、停牌和部分成交。
- Yahoo/Stooq 复权结果的独立公司行动对账。

这些 warning 不阻止受限研究，但必须阻止机构级、实盘级或可靠 alpha 声明。

## 工程保障

- `uv.lock` 锁定依赖及制品哈希。
- 研究输入按 SHA-256 写入 `data/archive/research_inputs/` 内容寻址归档。
- GitHub Actions 自动执行 Ruff、编译、数据契约、研究可信度、端到端复算和
  全量测试。
- 研究运行身份不再只依赖 Git HEAD；clean commit 或完整工作区补丁均可追溯。
