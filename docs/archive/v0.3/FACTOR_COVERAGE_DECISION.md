# 因子覆盖率决策

更新日期：`2026-06-13`

## 决策

从生产策略 `multifactor_balanced_us` 中移除：

- `debt_to_equity`
- `gross_margin`
- `operating_margin`

质量组保留 `roe` 和 `accruals`，按原相对权重重新归一为 `0.625/0.375`。

## 证据

运行 `20260613T103622Z-backtest-90ef83a1` 的平均覆盖率为：

| 因子 | 覆盖率 |
| --- | ---: |
| `debt_to_equity` | 68.88% |
| `gross_margin` | 41.37% |
| `operating_margin` | 70.96% |

三者均低于生产策略的 `80%` 覆盖门槛，并被 bias review 列为主要关注项。

## 原则

- 不使用未来值、横截面中位数或 0 填充来伪造覆盖率。
- 因子实现和 metadata 保留，供数据源升级或专项实验使用。
- 生产回测只输出当前配置中的因子，避免已排除因子继续进入诊断和审查。

## 重新纳入条件

只有在真实 PIT 数据下达到至少 80% 的平均覆盖率，并通过标签泄漏检查、因子诊断
和完整回测验收后，才可重新加入生产策略。
